# AI 使用说明

本项目由 AI 辅助完成，符合“AI 驱动的开发实践”任务要求。

## AI 完成的内容

1. 根据作业要求确定项目主题：C++ 2048 终端小游戏。
2. 根据参考仓库的结构设计项目目录。
3. 生成 C++17 游戏逻辑代码。
4. 生成 FTXUI 界面渲染代码。
5. 生成 CMake 构建文件。
6. 生成 README.md 文档。

## 人工需要完成的内容

1. 在本地编译运行并截图留档。
2. 将项目上传到个人 GitHub 仓库。
3. 根据课程要求提交个人仓库链接。

## 与参考仓库的关系

参考仓库 `Dashark/Tetris-mini-game` 是基于 FTXUI 的俄罗斯方块终端小游戏。本项目只参考其“C++ + CMake + FTXUI + src 目录”的工程组织方式，没有复制其游戏逻辑代码。
